### 🚀 Featured Projects

#### 🔷 Mirrors Hub
A Lua-based tooling hub with a **WindUI** interface, persistent config system, multi-language support, and an **AI-powered support assistant** built in — serverless backend on Vercel (`api/ask.js`) proxying requests from the in-game client to an AI model.

**Technical highlights:**
- 🌐 Serverless API (Node.js + Vercel) connecting the Lua client to an AI model
- 🎨 UI built with WindUI
- 💾 Profile/config system (save/load/export/import)
- 🌍 Multi-language support (EN/PT/ES)
