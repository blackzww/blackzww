## 🧭 About Me

I build tooling for **Roblox**, focused on **Lua scripting**, custom UI systems, and serverless backends that connect game clients to real infrastructure — configs, AI assistants, logging, and more.

Currently developing **Mirrors Hub**, a Lua tooling hub with:
- 🎨 Interface built on **WindUI**
- 🌐 **AI-powered support assistant**, backed by a Vercel serverless API (`api/ask.js`)
- 💾 Full config system — save / load / export / import
- 🌍 Multi-language support (EN / PT / ES)
